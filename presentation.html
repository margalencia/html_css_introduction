Текстовые редакторы и IDE

  • Notepad++
  • Brackets
  • Atom
  • Sublime Text

IDE (англ. Integrated development environment) — интегрированная среда разработки, включает в себя: текстовый редактор, компилятор и/или интерпретатор, средства автоматизации сборки, отладчик, средства для интеграции с системами контроля версий и т.д.

  • WebStorm, бесплатная лицензия для студентов и преподавателей и для оснащения учебных классов

Инструменты разработчика в браузерах

  • Chrome DevTools
  • Firefox Developer Tools
  • Microsoft Edge Developer Tools

Запускаются в браузере по нажатию на F12
